# Run the API (dev)

How to start the app locally with the FastAPI CLI.

---

## 1. Activate the virtual environment

```bash
. .venv/bin/activate
```

## 2. Run FastAPI in development mode

**Run from the repository root** (the folder that contains `app/`), not from inside `app/`. Imports use the `app` package (`from app.db import …`), so the project root must be the current working directory.

```bash
cd /path/to/fastapi-postgresql-crud
fastapi dev app/main.py
```

Auto reload

```bash
cd /path/to/fastapi-postgresql-crud
uvicorn app.main:app --reload
```

The CLI watches files and reloads on change.

If you run `fastapi dev main.py` **after** `cd app`, Python cannot resolve the `app` package and you will see `Import error: No module named 'app'`.
