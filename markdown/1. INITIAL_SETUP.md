# Initial setup (`fastapi-postgresql-crud`)

Step-by-step notes for creating the Python environment and a minimal FastAPI entrypoint.

---

## Create virtual environment

```bash
python3 -m venv .venv
```

## Activate it

```bash
. .venv/bin/activate
```

## Add `.gitignore`

- Include `.venv`.

## Install `pip`

```bash
pip install --upgrade pip
```

## Install `fastapi`

- You can install extras by listing them in the brackets.

```bash
pip install 'fastapi[standard]'
```

## Add `requirements.txt`

```bash
pip freeze > requirements.txt
```

## Install packages (requirements)

- Add the package in `requirements.txt`, then:

```bash
pip install -r requirements.txt
```

- Or install the package first, then refresh `requirements.txt`:

```bash
pip install package_name
pip freeze > requirements.txt
```

## If no repository

```bash
git init
git add .
git commit -m "initial commit"
```

## Create `main.py`

```bash
touch main.py
```

## Update `main.py`

```python
from fastapi import FastAPI

app = FastAPI()


@app.get("/")
def read_root():
    return {"message": "Hello World"}
```

## Swagger and OpenAPI

- **Swagger UI:** `http://localhost:8000/docs`
- **OpenAPI JSON:** `http://localhost:8000/openapi.json`
